## Cursor

Use cursor utilities to change the mouse cursor style of an element.

<div className="bd-example">
  <button className="btn btn-primary cursor-pointer">Pointer cursor</button>
  <button className="btn btn-secondary cursor-default">Default cursor</button>
  <button className="btn btn-success cursor-move">Move cursor</button>
  <button className="btn btn-danger cursor-not-allowed">Not allowed cursor</button>
</div>
