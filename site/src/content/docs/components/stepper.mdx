## Examples

### Basic

Steppers are built with CSS Grid and `<ol>` elements. By default, steps are displayed vertically. You can transform them into horizontal lists with responsive modifier classes.

{{< example >}}
<ol class="stepper">
  <li class="stepper-item active">Create account</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

### Gap

Customize the gap with styles that override the `--bs-stepper-gap` CSS variable.

{{< example >}}
<ol class="stepper" style="--bs-stepper-gap: 3rem">
  <li class="stepper-item active">Create account</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

## Horizontal

Use the `.stepper-horizontal{-sm|-md|-lg|-xl|-xxl}` classes to display steps horizontally starting at the specified breakpoint size. Horizontal steppers are rendered with `display: inline-grid` so their alignment can be modified (see below).

Here's the same stepper with modifier classes for every breakpoint.

{{< example >}}
<select class="form-select form-select-sm w-auto mb-4" id="stepper-switcher">
{{< stepper.inline >}}
{{- range $.Site.Data.breakpoints -}}
  <option value="stepper-horizontal{{ .abbr }}">horizontal{{ .abbr }}</option>
{{- end -}}
{{< /stepper.inline >}}
</select>

<ol class="stepper stepper-horizontal" id="stepper">
  <li class="stepper-item active">Create account</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

<script>
  const stepperSwitcher = document.getElementById('stepper-switcher');
  const stepper = document.getElementById('stepper');

  stepperSwitcher.addEventListener('change', function() {
    const selectedOption = stepperSwitcher.value;
    stepper.className = `stepper ${selectedOption}`;
  });
</script>

### Alignment

Use [text alignment utilities]({{< docsref "/utilities/text#text-alignment" >}}) (because we use `display: inline-grid`) to align the steps. The inline grid arrangement allows us to keep the steps equal width and ensures the connecting lines are rendered correctly.

{{< example >}}
<ol class="stepper stepper-horizontal">
  <li class="stepper-item active">Default stepper</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

{{< example class="text-center" >}}
<ol class="stepper stepper-horizontal">
  <li class="stepper-item active">Center stepper</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

{{< example class="text-end" >}}
<ol class="stepper stepper-horizontal">
  <li class="stepper-item active">End stepper</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

Apply `.w-100` to the stepper to make it full width. Stepper items will be stretched to fill the available space. Alignment doesn't affect full-width steppers.

{{< example >}}
<ol class="stepper stepper-horizontal w-100">
  <li class="stepper-item active">Create account</li>
  <li class="stepper-item active">Confirm email</li>
  <li class="stepper-item">Update profile</li>
  <li class="stepper-item">Finish</li>
</ol>
{{< /example >}}

### With anchors

Use anchor elements to build your stepper if it links across multiple pages. Add `role="button"` or use `<button>` elements if you're linking across sections in the same document.

Consider using our [link helpers]({{< docsref "/helpers/colored-links" >}}) for quick color control.

{{< example >}}
<div class="stepper">
  <a href="#" role="button" class="stepper-item link-body-emphasis active">Create account</a>
  <a href="#" role="button" class="stepper-item link-body-emphasis active">Confirm email</a>
  <a href="#" role="button" class="stepper-item link-secondary">Update profile</a>
  <a href="#" role="button" class="stepper-item link-secondary">Finish</a>
</div>
{{< /example >}}
